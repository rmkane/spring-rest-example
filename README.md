# Spring REST Example

Derived from: ["Spring REST Hello World Example"][mkyong-spring-boot]

## Build

```shell
mvn clean install
```

## Format

```shell
mvn com.coveo:fmt-maven-plugin:format
```

## Run

```shell
mvn spring-boot:run
```

## Swagger UI

Visit: http://localhost:8080/swagger-ui.html
